## 段落标签

::: info 段落标签
在网页中，要把文字有条理的显示出来，就需要将文字分段显示，在html中，`<p></p>`标签可以用于定义段落， 可以将网页分为若干个段落；
:::

使用了段落标签，可以将文档分割为多个层次。一眼看上去更有条理，`切记：不要忘了结束标签哦1`

``` html
<p>这里是段落...</p>
```

## 折行标签

如果您希望在不产生一个新起点的情况下进行换行（新行），请使用`<br/>`标签。

折行标签标签定义为一个换行符，它可以理解为简单的输入一个空行，而不是用来对内容进行拆分：

``` html
<br/>
```
